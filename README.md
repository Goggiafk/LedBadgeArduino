# CyberBadge Arduino Hardware  

[![Visit the website](https://img.shields.io/badge/Visit-Website-blue?logo=internet-explorer&logoColor=white&style=for-the-badge)](https://www.cyberbadge.net/)

Controller application for my electronics project conducted for Pixel Ant Games: Portable, rechargable and fully programmable LED display for parties and marketing purposes.

More info on: https://www.cyberbadge.net/

Made in: Arduino/C++
